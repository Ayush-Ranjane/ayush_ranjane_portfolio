## ⚡ Why Neo-Brutalism?

Neo-brutalism is characterized by bold, high-contrast aesthetics, unapologetic typography, stark borders, and vibrant colors. This portfolio rejects the "clean and minimal" norm, opting instead for a raw, engaging, and highly memorable user experience. 

## ✨ Key Features

- **🎨 Striking Visuals:** Hard shadows, bold borders, and vibrant accents that stand out.
- **🚀 Blazing Fast:** Powered by Next.js App Router for optimal performance.
- **🎭 Liquid Animations:** Smooth page transitions, scroll reveals, and micro-interactions using Framer Motion.
- **📱 Fully Responsive:** Carefully crafted layouts that look perfect on any device size.
- **🧩 Component-Driven:** Built using a rich, modular architecture with `shadcn/ui` and `@base-ui/react`.
- **📈 Analytics Ready:** Pre-configured with Vercel Analytics and Speed Insights.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library:** [React](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation:** [Framer Motion](https://www.framer.com/motion/) & `tw-animate-css`
- **UI Primitives:** [shadcn/ui](https://ui.shadcn.com/) & `@base-ui/react`
- **Deployment:** [Vercel](https://vercel.com/)

## 🏁 Getting Started

Want to run this project locally or use it as a template? Follow these simple steps.

### Prerequisites

You'll need Node.js (v18+) and your package manager of choice (npm, pnpm, or yarn).

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ayush-Ranjane/ayush_ranjane_portfolio.git
   cd ayush_ranjane_portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. **View the project**
   Open [http://localhost:3000](http://localhost:3000) in your browser to see the portfolio in action.
